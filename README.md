# 운동로그

매일 적는 운동 기록을 붙여넣으면 유산소/무산소를 나누고, 몸무게 기준으로 대략적인 칼로리를 계산하는 웹앱입니다.

## 기능

- 몸무게와 키 저장
- 운동 기록 붙여넣기
- 유산소/무산소 자동 분류
- 오늘 예상 소비 칼로리 계산
- 이번 달 운동한 날짜 수 계산
- 운동별 주의 메모 표시
- 날짜별 기록 저장과 삭제
- 백업 내보내기/가져오기

## GitHub Pages에 올리기

새 GitHub 저장소를 만들고 이 폴더 안의 파일들을 저장소 맨 위에 올립니다.

- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
- `assets` 폴더

그 다음 `Settings > Pages`에서 `Deploy from a branch`, `main`, `/root`를 선택하면 됩니다.

칼로리는 MET 기반의 추정치라 실제 기기 측정값과 다를 수 있습니다.
